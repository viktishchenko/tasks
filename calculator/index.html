Instruction to cross-check

Примеры вычислений для проверки работоспособности калькулятора и его соответствия критериям

Примеры ориентировочные, числа могут быть другими. Каждое правильно выполненное вычисление оцениваем в 2 балла. Если вычисления выполняются на лету и результат выводится без клика по кнопке "равно", это не является ошибкой.

Базовая функциональность +10

  1. 1+2=3
  2. 23+69.5=92.5
  3. 74*3-5=217
  4. 2+3=5 продолжаем ввод 4=4 - после равно следующая цифра перезаписывает результат
  5. есть кнопка, позволяющая очистить результат

Дополнительные математические операции +10

  1. 25√=5
  2. 9√+1=4
  3. 2xn2=4
  4. 15xn3=3375
  5. 10.1xn3=1030.301

Действия с отрицательными числами +10

  1. 9±÷3±=3 (-9 ÷ -3 => 3)
  2. 2+2±=0 (2 + -2 => 0)
  3. 2÷2±=-1 (2 ÷ -2 => -1)
  4. 9±xn3=-729
  5. 9±√=Error уведомление об ошибке

Действия с дробями +10

  1. 0.1+0.2=0.3
  2. 0.4-0.1=0.3
  3. 0.0004+0.0004=0.0008
  4. 0.1±*0.2=-0.02 (-0.1 * 0.2 => -0.02)
  5. 0.15±+0.15±=-0.3 (-0.15 + -0.15 => -0.3)
